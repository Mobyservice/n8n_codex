# Как получить chat_id (без публикации токена)

## Личка (owner)
1) Откройте Telegram
2) Найдите @userinfobot
3) Нажмите Start — он покажет ваш chat_id

## Групповой чат "Задачи"
1) Добавьте вашего бота в чат (и дайте право писать)
2) Напишите сообщение в чат
3) Откройте в браузере:
   https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates
4) Найдите объект ""chat"": { ""id"": -100... , ""title"": ""..."" }
   Это и есть TELEGRAM_TASKS_CHAT_ID
